# 🎬 Entertainment APIs (movies, TV, music, anime)

Media metadata and discovery.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| TMDB | Movies & TV metadata, images, credits | apiKey | Yes | https://developer.themoviedb.org/docs |
| OMDb | Movie/series info & posters (IMDb data) | apiKey | Yes | https://www.omdbapi.com/ |
| TVMaze | TV show schedules, episodes, cast | No | Yes | https://www.tvmaze.com/api |
| Watchmode | Streaming availability across services | apiKey | Yes | https://api.watchmode.com/ |
| Spotify Web API | Tracks, artists, playlists, audio features | OAuth | Yes | https://developer.spotify.com/documentation/web-api |
| Deezer | Music catalog & charts | No/OAuth | Yes | https://developers.deezer.com/api |
| Last.fm | Music metadata, scrobbles, charts | apiKey | Yes | https://www.last.fm/api |
| MusicBrainz | Open music encyclopedia | No | Yes | https://musicbrainz.org/doc/MusicBrainz_API |
| Genius | Song lyrics metadata & annotations | OAuth | Yes | https://docs.genius.com/ |
| AudioDB | Music artists, albums, artwork | apiKey | Yes | https://www.theaudiodb.com/api_guide.php |
| Jikan (MyAnimeList) | Unofficial MyAnimeList anime/manga | No | Yes | https://docs.api.jikan.moe/ |
| AniList | Anime & manga (GraphQL) | No/OAuth | Yes | https://anilist.gitbook.io/anilist-apiv2-docs/ |
| Kitsu | Anime & manga catalog | No/OAuth | Yes | https://kitsu.docs.apiary.io/ |
| TVDB | TV & movie metadata | apiKey | Yes | https://thetvdb.github.io/v4-api/ |
| Podcast Index | Open podcast directory & search | apiKey | Yes | https://podcastindex-org.github.io/docs-api/ |
