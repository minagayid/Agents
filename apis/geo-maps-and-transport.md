# 🗺️ Geo, maps & transportation APIs

Geocoding, places, routing, and transit data.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| OpenStreetMap Nominatim | Free geocoding & reverse geocoding | No | Yes | https://nominatim.org/release-docs/latest/api/Overview/ |
| Google Maps Platform | Geocoding, places, routes, tiles | apiKey | Yes | https://developers.google.com/maps |
| Mapbox | Maps, geocoding, navigation, tiles | apiKey | Yes | https://docs.mapbox.com/api/ |
| HERE | Maps, routing, geocoding, traffic | apiKey | Yes | https://developer.here.com/ |
| OpenCage | Forward/reverse geocoding aggregator | apiKey | Yes | https://opencagedata.com/api |
| IP-API | Geolocation from IP address | No | Yes | https://ip-api.com/ |
| ipinfo.io | IP geolocation & ASN data | apiKey | Yes | https://ipinfo.io/developers |
| ipgeolocation.io | IP geolocation, timezone, astronomy | apiKey | Yes | https://ipgeolocation.io/ |
| REST Countries | Country data (borders, currencies, flags) | No | Yes | https://restcountries.com/ |
| GeoNames | Geographical database (places, postal) | apiKey (user) | Yes | https://www.geonames.org/export/web-services.html |
| Zippopotam.us | Postal/ZIP code → place lookup | No | Yes | https://api.zippopotam.us/ |
| OpenRouteService | Routing, isochrones, matrix | apiKey | Yes | https://openrouteservice.org/dev/ |
| TransitLand | Aggregated public-transit data (GTFS) | apiKey | Yes | https://www.transit.land/documentation/ |
| Transport for London (TfL) | London transport status & journeys | apiKey | Yes | https://api-portal.tfl.gov.uk/ |
| OpenSky Network | Live & historical air traffic (ADS-B) | No/apiKey | Yes | https://openskynetwork.github.io/opensky-api/ |
| AviationStack | Real-time & historical flight data | apiKey | Yes | https://aviationstack.com/ |
| Amadeus for Developers | Flights, hotels, travel search | OAuth | Yes | https://developers.amadeus.com/ |
| NASA SSD/CNEOS | Solar system, orbits, close approaches | No | Yes | https://ssd-api.jpl.nasa.gov/ |
