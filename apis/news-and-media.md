# 📰 News & media APIs

Headlines, articles, and content feeds.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| NewsAPI.org | Headlines & articles from many sources | apiKey | Yes | https://newsapi.org/docs |
| GNews | Search & top headlines | apiKey | Yes | https://gnews.io/docs/ |
| The Guardian | Full Guardian content platform | apiKey | Yes | https://open-platform.theguardian.com/documentation/ |
| New York Times | Articles, books, movies, most-popular | apiKey | Yes | https://developer.nytimes.com/ |
| Currents | News from blogs & sources by topic | apiKey | Yes | https://currentsapi.services/en/docs/ |
| Mediastack | Live & historical news data | apiKey | Yes | https://mediastack.com/documentation |
| NewsData.io | News & crypto news, many languages | apiKey | Yes | https://newsdata.io/documentation |
| Hacker News (Firebase) | HN stories, comments, users | No | Yes | https://github.com/HackerNews/API |
| Reddit | Posts, comments, subreddits | OAuth | Yes | https://www.reddit.com/dev/api/ |
| Spaceflight News | Space news articles & reports | No | Yes | https://api.spaceflightnewsapi.net/ |
| RSS to JSON (rss2json) | Convert any RSS feed to JSON | No/apiKey | Yes | https://rss2json.com/docs |
| Wikipedia / MediaWiki | Article content, search, summaries | No | Yes | https://www.mediawiki.org/wiki/API:Main_page |
| Wikidata | Structured knowledge base | No | Yes | https://www.wikidata.org/wiki/Wikidata:Data_access |
