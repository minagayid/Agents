# ⚽ Sports APIs

Scores, fixtures, standings, and stats.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| API-FOOTBALL (API-Sports) | Football/soccer fixtures, stats, odds | apiKey | Yes | https://www.api-football.com/documentation-v3 |
| TheSportsDB | Free multi-sport metadata & scores | No/apiKey | Yes | https://www.thesportsdb.com/api.php |
| Football-Data.org | Football competitions & matches | apiKey | Yes | https://www.football-data.org/documentation/quickstart |
| balldontlie | NBA stats & games | No/apiKey | Yes | https://www.balldontlie.io/ |
| NHL (statsapi) | NHL scores, standings, stats | No | Yes | https://gitlab.com/dword/nhl-api-reference |
| MLB StatsAPI | MLB scores & stats | No | Yes | https://statsapi.mlb.com/docs/ |
| ESPN (hidden) | Multi-sport scoreboards (community) | No | Yes | https://github.com/pseudo-r/Public-ESPN-API |
| Sportradar | Comprehensive pro sports data | apiKey | Yes | https://developer.sportradar.com/ |
| API-Sports (multi) | Basketball, baseball, F1, rugby, etc. | apiKey | Yes | https://api-sports.io/documentation |
| Ergast (F1, archived) | Historical Formula 1 data | No | Yes | https://ergast.com/mrd/ |
| OpenF1 | Real-time & historical Formula 1 | No | Yes | https://openf1.org/ |
| Strava | Athlete activities & segments | OAuth | Yes | https://developers.strava.com/ |
| Wger | Workout & fitness manager | No/apiKey | Yes | https://wger.de/en/software/api |
