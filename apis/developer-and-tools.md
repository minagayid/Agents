# 🧰 Developer & utility APIs

Dev platforms, package registries, and everyday building blocks.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| GitHub REST | Repos, issues, PRs, users, actions | OAuth/token | Yes | https://docs.github.com/rest |
| GitLab | Repos, pipelines, issues | OAuth/token | Yes | https://docs.gitlab.com/ee/api/ |
| npm registry | Package metadata & downloads | No | Yes | https://github.com/npm/registry/blob/main/docs/REGISTRY-API.md |
| PyPI (JSON) | Python package metadata | No | Yes | https://warehouse.pypa.io/api-reference/json.html |
| crates.io | Rust package registry | No | Yes | https://crates.io/data-access |
| Packagist | PHP package registry | No | Yes | https://packagist.org/apidoc |
| Docker Hub | Image repositories & tags | No/token | Yes | https://docs.docker.com/docker-hub/api/latest/ |
| ipify | Get your public IP | No | Yes | https://www.ipify.org/ |
| httpbin | HTTP request & response testing | No | Yes | https://httpbin.org/ |
| JSONPlaceholder | Fake REST API for prototyping | No | Yes | https://jsonplaceholder.typicode.com/ |
| ReqRes | Hosted sample REST for testing | No | Yes | https://reqres.in/ |
| QR Server | Generate & read QR codes | No | Yes | https://goqr.me/api/ |
| screenshotone / thum.io | URL → screenshot | apiKey | Yes | https://www.thum.io/documentation |
| Cloudinary | Image/video upload & transform | apiKey | Yes | https://cloudinary.com/documentation |
| URLhaus / urlscan.io | URL & malware scanning | No/apiKey | Yes | https://urlscan.io/docs/api/ |
| Have I Been Pwned | Breached-account & password checks | apiKey | Yes | https://haveibeenpwned.com/API/v3 |
| Shortlink (is.gd / TinyURL) | URL shortening | No/apiKey | Yes | https://tinyurl.com/app/dev |
| Public Holidays (Nager.Date) | Holidays by country/year | No | Yes | https://date.nager.at/Api |
| WorldTimeAPI | Current time & timezone data | No | Yes | http://worldtimeapi.org/ |
| TimeZoneDB | Timezone & DST lookup | apiKey | Yes | https://timezonedb.com/api |
