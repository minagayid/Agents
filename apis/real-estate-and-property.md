# 🏠 Real estate & property APIs

Listings, valuations, and property data.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| ATTOM Data | US property, sales, AVM, deeds | apiKey | Yes | https://api.developer.attomdata.com/docs |
| Realtor / Realty (RapidAPI) | US listings & property details | apiKey | Yes | https://rapidapi.com/apidojo/api/realty-in-us |
| Zillow (Bridge/RapidAPI) | Listings & Zestimate (via partners) | apiKey | Yes | https://www.bridgeinteractive.com/developers/zillow-group-data/ |
| Rentcast | US property data, AVM, rent estimates | apiKey | Yes | https://developers.rentcast.io/ |
| Estated | US property records & valuations | apiKey | Yes | https://estated.com/developers/docs |
| Melissa Property | Property & address data | apiKey | Yes | https://www.melissa.com/developer/property |
| PropertyData (UK) | UK property market data | apiKey | Yes | https://propertydata.co.uk/api |
| Rapid Realtor (Idealista/EU) | European listings (via partners) | apiKey | Yes | https://developers.idealista.com/access-request |
| HUD USER | US housing & urban development data | No | Yes | https://www.huduser.gov/portal/dataset/fmr-api.html |
| WalkScore | Walkability, transit & bike scores | apiKey | Yes | https://www.walkscore.com/professional/api.php |
| US Census (ACS housing) | Housing & demographic data | apiKey | Yes | https://www.census.gov/data/developers/data-sets.html |
