# 👥 Social & productivity APIs

Social platforms, calendars, docs, and productivity tools.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| Reddit | Posts, comments, subreddits | OAuth | Yes | https://www.reddit.com/dev/api/ |
| X (Twitter) | Tweets, users, timelines | OAuth | Yes | https://developer.twitter.com/en/docs |
| Mastodon | Federated social (per-instance) | OAuth | Yes | https://docs.joinmastodon.org/api/ |
| Bluesky (AT Protocol) | Decentralized social | OAuth/token | Yes | https://docs.bsky.app/ |
| YouTube Data | Videos, channels, playlists, search | apiKey/OAuth | Yes | https://developers.google.com/youtube/v3 |
| Twitch | Streams, users, clips | OAuth | Yes | https://dev.twitch.tv/docs/api/ |
| LinkedIn | Profiles, posts, share (partner) | OAuth | Yes | https://learn.microsoft.com/linkedin/ |
| Notion | Pages, databases, blocks | OAuth/token | Yes | https://developers.notion.com/ |
| Airtable | Bases, tables, records | token | Yes | https://airtable.com/developers/web/api/introduction |
| Google Calendar | Events & calendars | OAuth | Yes | https://developers.google.com/calendar/api |
| Todoist | Tasks & projects | OAuth/token | Yes | https://developer.todoist.com/rest/v2/ |
| Trello | Boards, cards, lists | apiKey/OAuth | Yes | https://developer.atlassian.com/cloud/trello/ |
| Calendarific | Global public holidays | apiKey | Yes | https://calendarific.com/api-documentation |
| Nager.Date | Free public-holiday data | No | Yes | https://date.nager.at/Api |
| WorldTimeAPI | Current time & timezones | No | Yes | http://worldtimeapi.org/ |
| Google Sheets | Read/write spreadsheet data | OAuth | Yes | https://developers.google.com/sheets/api |
