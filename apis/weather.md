# 🌦️ Weather & climate APIs

Free/public weather, forecast, and climate data APIs. `Auth`: what's required to call it.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| Open-Meteo | Free weather forecast, historical & marine, no key for non-commercial | No | Yes | https://open-meteo.com/ |
| OpenWeatherMap | Current, forecast, historical weather worldwide | apiKey | Yes | https://openweathermap.org/api |
| WeatherAPI.com | Real-time, forecast, history, astronomy, air quality | apiKey | Yes | https://www.weatherapi.com/ |
| National Weather Service (US) | Official US forecasts & alerts | No | Yes | https://www.weather.gov/documentation/services-web-api |
| Met.no (Yr) | Norwegian Met Institute global forecasts | No | Yes | https://api.met.no/ |
| Tomorrow.io | Hyperlocal weather & climate insights | apiKey | Yes | https://www.tomorrow.io/weather-api/ |
| Visual Crossing | Weather history & forecast, timeline API | apiKey | Yes | https://www.visualcrossing.com/weather-api |
| WeatherKit (Apple) | Apple weather data | OAuth (JWT) | Yes | https://developer.apple.com/weatherkit/ |
| AccuWeather | Forecasts, indices, alerts | apiKey | Yes | https://developer.accuweather.com/ |
| Météo-France | French met service open data | apiKey | Yes | https://portail-api.meteofrance.fr/ |
| 7Timer! | Free astronomy & civil weather, no key | No | Yes | http://www.7timer.info/doc.php |
| OpenAQ | Open global air-quality measurements | apiKey | Yes | https://docs.openaq.org/ |
| AirVisual (IQAir) | Air quality & pollution | apiKey | Yes | https://www.iqair.com/air-pollution-data-api |
| Sunrise-Sunset | Sunrise/sunset times for a location | No | Yes | https://sunrise-sunset.org/api |
| Aurora (NOAA SWPC) | Space weather & aurora forecasts | No | Yes | https://www.swpc.noaa.gov/products-and-data |
