# 🍽️ Food & drink APIs

Recipes, ingredients, and beverages.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| Spoonacular | Recipes, ingredients, nutrition, meal plans | apiKey | Yes | https://spoonacular.com/food-api |
| TheMealDB | Free recipe & meal database | No/apiKey | Yes | https://www.themealdb.com/api.php |
| TheCocktailDB | Cocktail & drink recipes | No/apiKey | Yes | https://www.thecocktaildb.com/api.php |
| Edamam Recipe Search | Recipe search & diet filters | apiKey | Yes | https://developer.edamam.com/edamam-recipe-api |
| Open Food Facts | Open food-product database | No | Yes | https://world.openfoodfacts.org/data |
| Open Brewery DB | Breweries directory | No | Yes | https://www.openbrewerydb.org/documentation |
| PunkAPI (BrewDog) | Beer recipes & data | No | Yes | https://punkapi.com/documentation/v2 |
| Fruityvice | Fruit nutrition data | No | Yes | https://www.fruityvice.com/ |
| Tasty (RapidAPI) | BuzzFeed Tasty recipes | apiKey | Yes | https://rapidapi.com/apidojo/api/tasty |
| Wine (Global Wine Score) | Wine ratings & scores | apiKey | Yes | https://www.globalwinescore.com/ |
| Coffee (Sampleapis) | Sample coffee data for prototyping | No | Yes | https://sampleapis.com/api-list/coffee |
