# 🔬 Science & space APIs

Astronomy, physics, earth science, and research data.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| NASA APIs | APOD, Mars rovers, NEO, EPIC, imagery | apiKey (DEMO_KEY ok) | Yes | https://api.nasa.gov/ |
| Launch Library 2 | Rocket launches, events, spacecraft | No | Yes | https://thespacedevs.com/llapi |
| Open Notify | ISS location & people in space | No | Yes | http://open-notify.org/Open-Notify-API/ |
| SpaceX (r-spacex) | Launches, rockets, capsules (community) | No | Yes | https://github.com/r-spacex/SpaceX-API |
| USGS Earthquake | Real-time earthquake feeds | No | Yes | https://earthquake.usgs.gov/fdsnws/event/1/ |
| Solar System OpenData | Planets, moons, bodies | No | Yes | https://api.le-systeme-solaire.net/ |
| arXiv | Scientific paper metadata & search | No | Yes | https://info.arxiv.org/help/api/index.html |
| CrossRef | Scholarly metadata & DOIs | No | Yes | https://www.crossref.org/documentation/retrieve-metadata/rest-api/ |
| OpenAlex | Open catalog of scholarly works | No | Yes | https://docs.openalex.org/ |
| Semantic Scholar | Academic graph & paper data | No/apiKey | Yes | https://api.semanticscholar.org/ |
| CORE | Open-access research papers | apiKey | Yes | https://core.ac.uk/documentation/api |
| PubChem | Chemical compounds & properties | No | Yes | https://pubchem.ncbi.nlm.nih.gov/docs/pug-rest |
| GBIF | Global biodiversity occurrence data | No | Yes | https://www.gbif.org/developer/summary |
| iNaturalist | Species observations & taxa | No/OAuth | Yes | https://api.inaturalist.org/v1/docs/ |
| NOAA Climate Data | US climate & weather records | apiKey | Yes | https://www.ncdc.noaa.gov/cdo-web/webservices/v2 |
| ISS Notify / wheretheiss.at | Live ISS position & passes | No | Yes | https://wheretheiss.at/w/developer |
