# 🎮 Games & comics APIs

Video-game data, game platforms, and comics.

| API | Description | Auth | HTTPS | Docs |
|-----|-------------|------|-------|------|
| RAWG | Largest video-game database | apiKey | Yes | https://rawg.io/apidocs |
| IGDB | Games metadata (by Twitch) | OAuth | Yes | https://api-docs.igdb.com/ |
| CheapShark | PC game deals & price comparison | No | Yes | https://apidocs.cheapshark.com/ |
| FreeToGame | Free-to-play games list & details | No | Yes | https://www.freetogame.com/api-doc |
| Steam Web API | Steam users, games, stats | apiKey | Yes | https://steamcommunity.com/dev |
| PandaScore | Esports data & odds | apiKey | Yes | https://developers.pandascore.co/ |
| Pokémon (PokéAPI) | Pokémon data | No | Yes | https://pokeapi.co/ |
| Scryfall | Magic: The Gathering cards | No | Yes | https://scryfall.com/docs/api |
| Deck of Cards | Playing-card deck simulation | No | Yes | https://deckofcardsapi.com/ |
| Chess.com | Player, game, and club data | No | Yes | https://www.chess.com/news/view/published-data-api |
| Lichess | Open chess platform data | No/OAuth | Yes | https://lichess.org/api |
| Open Trivia DB | Free trivia questions | No | Yes | https://opentdb.com/api_config.php |
| Marvel | Marvel comics, characters, creators | apiKey | Yes | https://developer.marvel.com/ |
| Superhero API | Superhero stats & bios | apiKey | Yes | https://superheroapi.com/ |
| xkcd | Webcomic metadata | No | Yes | https://xkcd.com/json.html |
| Comic Vine | Comics metadata database | apiKey | Yes | https://comicvine.gamespot.com/api/ |
