# skills-pack (plugin)

A bundle of 10 ready-to-use engineering Agent Skills from this repo's
[`skills/library/`](../../skills/library/README.md).

## Install

```bash
/plugin marketplace add minagayid/Agents
/plugin install skills-pack@agents-collection
```

## Included skills

`conventional-commits` · `pr-description` · `code-review-checklist` · `unit-test-writer` ·
`dockerfile-author` · `rest-api-designer` · `readme-generator` · `changelog-keeper` ·
`sql-optimizer` · `debugging-methodology`

Each is a plain `SKILL.md`, so you can also copy any folder directly into your own
`.claude/skills/` (or any Agent Skills-compatible client) without the plugin.
