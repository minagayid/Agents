# 🛠️ Tools & Frameworks

Agent SDKs, coding agents, and orchestration frameworks for building and running agents.

## Anthropic

| Tool | Description | Install |
|------|-------------|---------|
| [claude-code](https://github.com/anthropics/claude-code) | Agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster. | `npm i -g @anthropic-ai/claude-code` |
| [claude-agent-sdk-python](https://github.com/anthropics/claude-agent-sdk-python) | Python SDK for building Claude agents. | `pip install claude-agent-sdk` |
| [claude-agent-sdk-typescript](https://github.com/anthropics/claude-agent-sdk-typescript) | TypeScript SDK for building Claude agents. | `npm i @anthropic-ai/claude-agent-sdk` |
| [claude-agent-sdk-demos](https://github.com/anthropics/claude-agent-sdk-demos) | Practical examples of Claude Agent SDK usage. | clone |
| [launch-your-agent](https://github.com/anthropics/launch-your-agent) | Take a concept to a live Claude Managed Agent deployment. | clone |
| [defending-code-reference-harness](https://github.com/anthropics/defending-code-reference-harness) | Skills + autonomous harness for threat modeling, scanning, triage, and patching. | clone |
