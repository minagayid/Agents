# 🔌 MCP Server Directory

A broad, categorized directory of Model Context Protocol servers. Names are drawn from the
[official reference servers](https://github.com/modelcontextprotocol/servers), the
[MCP Registry](https://github.com/modelcontextprotocol/registry), and the curated awesome‑lists
([wong2](https://github.com/wong2/awesome-mcp-servers) ·
[punkpeye](https://github.com/punkpeye/awesome-mcp-servers) ·
[appcypher](https://github.com/appcypher/awesome-mcp-servers)).

> ⭐ = maintained by the company/product itself (an "official" integration).
> For install commands of the most common servers, see [`mcp-servers.json`](mcp-servers.json).
> To find any server's repo/URL, search the [MCP Registry](https://github.com/modelcontextprotocol/registry)
> or the awesome‑lists above.
