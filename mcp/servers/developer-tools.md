Version control, project management, code intelligence, and API tooling. ⭐ = company-maintained.

| Server | Repo | Install |
|--------|------|---------|

| GitLab (reference, archived) | [servers-archived/gitlab](https://github.com/modelcontextprotocol/servers-archived/tree/main/src/gitlab) | `npx -y @modelcontextprotocol/server-gitlab` |
| Git (reference) | [modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers/tree/main/src/git) | `uvx mcp-server-git` |
| Atlassian Jira/Confluence | [sooperset/mcp-atlassian](https://github.com/sooperset/mcp-atlassian) | `uvx mcp-atlassian` |
| Linear ⭐ | [linear](https://linear.app/docs/mcp) | remote: `https://mcp.linear.app/sse` |
| Notion ⭐ | [makenotion/notion-mcp-server](https://github.com/makenotion/notion-mcp-server) | `npx -y @notionhq/notion-mcp-server` |
| JetBrains ⭐ | [JetBrains/mcp-jetbrains](https://github.com/JetBrains/mcp-jetbrains) | `npx -y @jetbrains/mcp-proxy` |

| shadcn/ui ⭐ | [shadcn-ui/ui](https://github.com/shadcn-ui/ui) | `npx shadcn@latest mcp` |
| Octocode (code search) | [bgauryy/octocode-mcp](https://github.com/bgauryy/octocode-mcp) | `npx -y octocode-mcp` |
