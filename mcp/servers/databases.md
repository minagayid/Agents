# MCP servers — Databases & data

Relational, NoSQL, vector, and warehouse access. ⭐ = company-maintained.

| Server | Repo | Install |
|--------|------|---------|
| PostgreSQL (reference, archived) | [servers-archived/postgres](https://github.com/modelcontextprotocol/servers-archived/tree/main/src/postgres) | `npx -y @modelcontextprotocol/server-postgres <conn>` |
| SQLite (reference, archived) | [servers-archived/sqlite](https://github.com/modelcontextprotocol/servers-archived/tree/main/src/sqlite) | `uvx mcp-server-sqlite --db-path <file>` |

| MongoDB ⭐ | [mongodb-js/mongodb-mcp-server](https://github.com/mongodb-js/mongodb-mcp-server) | `npx -y mongodb-mcp-server` |
| Redis ⭐ | [redis/mcp-redis](https://github.com/redis/mcp-redis) | `uvx --from git+https://github.com/redis/mcp-redis redis-mcp-server` |
