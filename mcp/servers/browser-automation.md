# MCP servers — Browser automation

Drive real/headless browsers, capture pages, and automate web UIs. ⭐ = company-maintained.

| Server | Repo | Install |
|--------|------|---------|
| Playwright ⭐ (Microsoft) | [microsoft/playwright-mcp](https://github.com/microsoft/playwright-mcp) | `npx -y @playwright/mcp@latest` |
| Puppeteer (reference, archived) | [servers-archived/puppeteer](https://github.com/modelcontextprotocol/servers-archived/tree/main/src/puppeteer) | `npx -y @modelcontextprotocol/server-puppeteer` |
| Browserbase ⭐ (cloud browsers) | [browserbase/mcp-server-browserbase](https://github.com/browserbase/mcp-server-browserbase) | `npx -y @browserbasehq/mcp` |
| Hyperbrowser ⭐ | [hyperbrowserai/mcp](https://github.com/hyperbrowserai/mcp) | `npx -y hyperbrowser-mcp` |
| browser-use | [browser-use/browser-use](https://github.com/browser-use/browser-use) | see repo |
| Browser MCP (control your own browser) | [browsermcp/mcp](https://github.com/browsermcp/mcp) | `npx -y @browsermcp/mcp` |
| Skyvern (workflow automation) | [Skyvern-AI/skyvern](https://github.com/Skyvern-AI/skyvern) | see repo |
| Notte ⭐ | [nottelabs/notte](https://github.com/nottelabs/notte) | see repo |
| Oxylabs Web Scraper ⭐ | [oxylabs/oxylabs-mcp](https://github.com/oxylabs/oxylabs-mcp) | `uvx oxylabs-mcp` |
| AgentQL ⭐ (structured extraction) | [tinyfish-io/agentql-mcp](https://github.com/tinyfish-io/agentql-mcp) | `npx -y agentql-mcp` |
