# 🧠 Skills Library

A broader library of Agent Skills — official first‑party collections, large community collections,
and the domains they cover. Skills are portable across Claude Code, Codex, Gemini CLI, Cursor, and
any client that supports the [Agent Skills standard](https://agentskills.io).

## First‑party collections

| Collection | What it covers | Install |
|------------|----------------|---------|
| [anthropics/skills](https://github.com/anthropics/skills) | Anthropic's official skills (docs, design, dev, comms). See [`README.md`](README.md) for the full list. | `/plugin marketplace add anthropics/skills` |
| [google/skills](https://github.com/google/skills) | Google Cloud & product skills (Gemini API, BigQuery, GKE, Cloud Run, Ads…). | `npx skills add google/skills` |
| [agentskills/agentskills](https://github.com/agentskills/agentskills) | The Agent Skills standard, spec, and reference. | see [agentskills.io](https://agentskills.io) |

> ⚠️ Community skills run instructions/scripts in your environment — read a `SKILL.md` (and any
> bundled scripts) before enabling it, the same way you'd review a dependency.

## Common skill domains

Where to look (or what to build) by area:

- **Documents & office** — `docx`, `pdf`, `pptx`, `xlsx`, doc co‑authoring, report generation
- **Design & front‑end** — `frontend-design`, `canvas-design`, `theme-factory`, brand guidelines, `algorithmic-art`
- **Engineering** — `mcp-builder`, `skill-creator`, `webapp-testing`, code review, refactoring, testing, git workflows
- **Data & analytics** — SQL authoring, BigQuery/warehouse helpers, dataviz, spreadsheet analysis
- **Cloud & DevOps** — GKE / Cloud Run / AWS deploy helpers, Terraform, CI/CD, incident response
- **Product & business** — PRDs, roadmaps, competitive research, marketing copy, `internal-comms`
- **Research & writing** — literature review, summarization, citation management, long‑form drafting
- **Security & compliance** — threat modeling, secure‑coding review, dependency audits
